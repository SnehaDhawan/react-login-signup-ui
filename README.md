# React Login & Signup UI

This project is a simple React-based Login and Signup UI created as part of a technical evaluation assignment.

## Features
- Login page with email & password validation
- Signup page with name, email, password validation
- Clean and modular React components
- Responsive UI
- No backend (UI-focused assignment)

## Technologies Used
- React.js
- JavaScript (ES6)
- HTML5 & CSS3

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Note
This project focuses only on frontend UI and validations. Backend APIs and database can be integrated in real-world applications.
